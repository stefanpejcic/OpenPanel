# How license expiration affects OpenPanel

When the Enterprise license expires, all OpenAdmin UI pages are redirected to a **“License Expired”** page, example:

![license expired openadmin](https://i.postimg.cc/Xj7ncdPm/Snimak-ekrana-napravlen-dana-2026-03-05-19-40-35.png)

This continues until the license key is removed. Once the key is removed, the admin panel automatically downgrades to the **Community Edition**, meaning premium features are no longer available.

The **OpenPanel UI** continues to function normally, and all features remain accessible to existing users.

To downgrade OpenPanel Enterprise to Community edition: [How-to Guides > Downgrade License](/docs/articles/license/how_to_downgrade_openpanel_license/)

**Summary**

- **License expired, key still present:** Admin panel is blocked; user panel works normally.  
- **License expired, key removed:** Admin panel downgrades to Community Edition features only.

Our goal is not to lock you into using the panel or increase prices year after year. We provide the **Community Edition** so you can familiarize yourself with the stack.

You can also start [a **30-day trial** to access premium features](/blog/openpanel-enterprise-free-trial/).

Finally, we offer a **fixed pricing guarantee**, so your subscription remains stable at [**€14.95/month** or **€149.50/year**](https://my.openpanel.com/index.php?rp=/store/openpanel/enterprise-license).
