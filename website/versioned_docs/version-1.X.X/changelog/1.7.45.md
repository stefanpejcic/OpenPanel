Released on March 06, 2026

### 💅 Polish
- [Hungarian translation](https://community.openpanel.org/d/262-goviris-magyar-hungarian-translation-for-openpanel)
- Added option to delete domains from *OpenAdmin > Domains* page.
- Enhanced MySQL user password restrictions: [all characters are now supported](/docs/articles/accounts/forbidden-usernames/#mysql--mariadb).
- Added 'SSH Allowlist' on *OpenAdmin > Notifications* page.
- Changing password for user from OpenAdmin, now terminates all sessions for the user.

- Fixed bug enabling maintenance_mode for WordPress from WP Manager.
- Fixed false-positive bug detecting maintenance_mode for WordPress in WP Manager, when `wp-content/maintenance.php` template is present.
