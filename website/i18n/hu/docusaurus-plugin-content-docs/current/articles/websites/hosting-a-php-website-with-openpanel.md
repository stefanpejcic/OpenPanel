# PHP-webhely üzemeltetése OpenPanel segítségével

Ez az útmutató végigvezeti Önt egy PHP-webhely OpenPanelen való üzemeltetésén.

---

## 1. lépés – Domain hozzáadása

1. Válassza az **OpenPanel → Domains → Add New Domain** menüpontot.
2. Adja meg új domain nevét, majd kattintson a **Domain hozzáadása** lehetőségre.

![domain hozzáadása](https://i.postimg.cc/qMTzYC1t/slika.png)

---

## 2. lépés – Állítsa be a PHP verzióját

1. Nyissa meg az **OpenPanel → PHP → PHP verzió kiválasztása** lehetőséget.
2. Domainjéhez kattintson a verzióra, majd a "Módosítás" gombra.

![php verzió módosítása](https://i.postimg.cc/Cwy4B9bL/slika.png)

---

## 3. lépés – Töltse fel fájljait

1. Nyissa meg az **OpenPanel → Fájlkezelő** lehetőséget.
2. Navigáljon a domain címtárához.
3. Kattintson a **Feltöltés** lehetőségre a fájlok kiválasztásához eszközéről, vagy válassza a **Letöltés URL-ről** lehetőséget, ha külső forrásból (például GitHubból) szeretne letölteni fájlokat.

![fájlok feltöltése](https://i.postimg.cc/c4xdw7Dc/slika.png)

---

## 4. lépés – Tesztelje webhelyét

1. Keresse fel domainjét a „https://” használatával az SSL-generálás elindításához.
2. A php webhelyednek most élőben kell lennie és azonnal láthatónak kell lennie.
