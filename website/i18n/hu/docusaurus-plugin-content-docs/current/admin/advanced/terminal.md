# Terminál

A webterminál az **OpenAdmin** alkalmazásban érhető el, így a kiemelt rendszergazdák közvetlenül a böngészőből kezelhetik szerverkörnyezetüket.

1. Jelentkezzen be az OpenAdmin panelre.
2. Lépjen a **Speciális > Terminál** elemre.
3. Használja a legördülő menüt vagy a választót a kívánt shell-típus kiválasztásához: "sh" vagy "bash".
4. A terminál interfész betöltődik, és azonnal elkezdheti a parancsok beírását.

Korlátozások:
- A webterminál nem interaktív.
- A fájlok feltöltése nem támogatott a terminálon keresztül.

:::info
Az OpenPanel 1.7.1-től kezdődően a webterminál csak a Super Admin számára érhető el. A többi adminisztrátori felhasználó többé nem férhet hozzá.
A webterminál teljes letiltásához, beleértve a Super Admin alkalmazást is, olvassa el a [A terminál funkció letiltása az OpenAdmin programban] című részt (/docs/articles/dev-experience/disable-openadmin-web-terminal/)
:::
