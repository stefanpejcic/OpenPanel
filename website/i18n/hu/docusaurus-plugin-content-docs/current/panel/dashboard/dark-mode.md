# Sötét mód

![dashboard_dark.png](/img/panel/v2/darkmode1.png)

Az OpenPanel téma beépített Sötét módot kínál, amely egyetlen kattintással aktiválható az oldalsáv bal alsó részén található opciók segítségével:

![Sötét mód kapcsolója](/img/panel/v2/darkmodes.gif)

Ha a Sötét mód aktiválva van, az az összes OpenPanel oldalra vonatkozik, beleértve a bejelentkezési oldalt is.

A Sötét mód beállításait elmenti a böngésző.
