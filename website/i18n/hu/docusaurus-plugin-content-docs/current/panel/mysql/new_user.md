# Felhasználó létrehozása

Egyszerűen, néhány kattintással hozhat létre új MySQL-felhasználót. A felhasználók az adatbázisokkal vagy a MySQL szerverrel való interakcióra szolgálnak.

Lépjen az **OpenPanel > MySQL > Felhasználó létrehozása** elemre:

1. **Adjon meg egy felhasználónevet**
Írja be a kívánt felhasználónevet a beviteli mezőbe. Győződjön meg arról, hogy a név egyedi, és követi a MySQL elnevezési szabályait (pl. kerülje a szóközöket vagy a speciális karaktereket).

2. **Adjon meg egy jelszót**
Írja be a kívánt jelszót a második beviteli mezőbe. Erős véletlenszerű jelszó generálásához kattintson a beviteli mező melletti gombra.

3. **Kattintson a "Felhasználó létrehozása" gombra.
Miután megadta vagy létrehozta a felhasználónevet és a jelszót, kattintson a **Felhasználó létrehozása** gombra az új MySQL-felhasználó létrehozásához.

---

## Bevált gyakorlatok

- Használjon rövid és leíró neveket (pl. "app_data", "wordpress_db").
- Kerülje a nagybetűk, szóközök vagy speciális karakterek használatát.
- Győződjön meg arról, hogy a felhasználónévnek értelme van a későbbi könnyebb azonosítás érdekében.
