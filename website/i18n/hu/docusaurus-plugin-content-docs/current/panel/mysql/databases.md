# Adatbázisok

![databases.png](/img/panel/v2/databases_main.png)

A MySQL-adatbázisok a webhely adatainak tárolására és kezelésére szolgálnak, mint például a tartalom, a felhasználói információk és a termékadatok, így elérhetővé és rendszerezhetővé teszik azokat a webalkalmazások számára.

Az Adatbázisok oldalon megtekintheti és kezelheti MySQL-adatbázisait.

Az oldalon található egy táblázat, amely tartalmazza a felhasználói adatbázisokat a hozzájuk kapcsolódó opciókkal együtt:

- **Adatbázisok:** Itt megtekintheti az adatbázisok méretét és az egyes adatbázisokhoz rendelt összes felhasználót.

Az Adatbázisok oldalon a következő lehetőségek állnak rendelkezésre:

- **Adatbázisok és hozzárendelt felhasználók megtekintése**
- **Új adatbázis létrehozása**
- **MySQL adatbázis törlése**

## Hozzon létre egy MySQL adatbázist

Új MySQL adatbázis létrehozásához kattintson az "Új adatbázis" gombra, és írja be az új adatbázis nevét:

![databases_new.png](/img/panel/v2/databases_new.png)

## Töröljön egy MySQL adatbázist

Meglévő MySQL adatbázis törléséhez kattintson a táblázatban az adatbázis neve melletti "Törlés" gombra.

Ezután kattintson ugyanarra a „Megerősítés” gombra.

![databases_delete_db.png](/img/panel/v2/databases_del.png)

:::veszély
⚠️ Egy MySQL-adatbázis törlésével az adatbázishoz tartozó összes tábla és adat véglegesen törlődik.
:::
