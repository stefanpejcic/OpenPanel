# Szolgáltatások

A Szolgáltatások oldalon megtekintheti a szolgáltatások aktuális valós idejű erőforrás-használatát, és kezelheti azokat.

:::info
A Szolgáltatások oldal csak akkor jelenik meg, ha a „szolgáltatások” modul engedélyezve van a szerveren, és szolgáltatásként szerepel a tárhelytervben.
A szolgáltatások elindításához vagy leállításához használja a Docker funkciót (ha elérhető), vagy forduljon tárhelyszolgáltatójához.
:::

Egy szolgáltatás kezeléséhez válassza ki a nevét a kiválasztási lehetőségek közül.

Az egységes szolgáltatási oldalon megtekintheti:

- aktuális szolgáltatási állapot
- erőforrás-használat (CPU, memória, lemez I/O, PID-k...)
- konténer neve (más konténerek szolgáltatásához való csatlakozáshoz használandó)
- opciók kezelése: engedélyezés/letiltás
- konténernaplók
